Quadra Quest

About the Quadra Quest

This is a Quadra Quest board game designed for four players, where each player competes to reach the finish line at position 99. The game involves rolling dice to determine the number of steps a player can move their token. The game board includes special cells such as black holes and safe zones that add an element of strategy and chance.

Objective

The objective of the game is to be the first player to move your token from the start (position 0) to the finish line (position 99).

Game Setup

  • Player Names: Enter the names of the players in the input fields provided for Player 1, Player 2, Player 3, and Player 4.
  • Start Game: Click the "Start Game" button to initialize the game. Players with entered names will be able to participate, and their respective roll buttons will be enabled in turn.

Game Board

The board consists of 100 cells arranged in a 10x10 grid. The cells are numbered from 0 (Start) to 99 (Finish).

  • Black Holes (Red Cells): Landing on these cells can send you back to the start unless you are in a safe zone.
  • Safe Zones (Green Cells): Landing on these cells provides immunity from black holes.

Rules

  1. Rolling the Dice: Players take turns rolling the dice by clicking their respective "Roll" button. The player to start the game is Player 1, followed by Player 2, Player 3, and Player 4 in cyclic order.
  2. Movement: A dice roll results in a number between 1 and 6. The token moves forward by the number rolled on the dice. If the move lands the token on a black hole, the token returns to the start unless the token is in a safe zone. Landing on a safe zone marks the token as immune to black holes for future turns.
  3. Winning the Game: The first player to reach or exceed position 99 wins the game. The game announces the winner and disables further rolls.
  4. Game Flow: After each roll, the turn moves to the next player. Players must roll a number that allows their token to land within the board limits (positions 0 to 99). The status message updates with each player's roll result and token movement.

Special Cells

  • Black Holes: If a player lands on a cell numbered 7, 12, 25, 40, 50, 60, or 94, the player moves back to the start unless they are in a safe zone.
  • Safe Zones: Cells numbered 2 and 38 provide immunity from black holes for future moves.

Enjoy the game and may the best strategist win!
